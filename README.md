変更点: ミニゲーム「浮遊回避」での障害物の間隔を広くしました。

- ファイル: `highlow/highlow.js`
- 変更: spawnObstacle() の gap を 90 -> 130 に増加

目的: プレイヤーが障害物の間を回避しやすくするため。

動作確認手順:
1. `index.html` をブラウザで開く（ローカルファイルでOK）。
2. ゲームを開始し、ミニゲームを解放してから「ミニゲーム」ボタンでパネルを開く。
3. 「Start」を押してミニゲームを開始し、障害物の間隔が広くなっていることを確認する。

補足: ギャップの値は `highlow/highlow.js` の `spawnObstacle()` 内の `gap` で調整できます。

追加仕様: 30秒を超えると徐々にギャップが狭くなります（30s で 130px、60s で 90px に線形で縮小）。

自動クリックの解放: 合計クリック数が1000回に達すると自動クリック機能が解放されます。解放後は1秒ごとに現在の1クリック分の金額が自動で加算されます（開始時に自動で有効化）。
